# zerotier-ui

[![Build](https://github.com/gsmlg-ci/zerotier-ui/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/zerotier-ui/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/zerotier-ui`
- `ghcr.io/gsmlg-dev/zerotier-ui`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/zerotier-ui:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/zerotier-ui:latest
```

## Build

```bash
docker build -t gsmlg/zerotier-ui .
```
